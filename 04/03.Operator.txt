13. What is the difference between `typeof` and `instanceof`?
The typeof operator return a string which indicates the type of operand's value but instanceof return boolean value which tells us weather a object is an instance of a class or not.
typeof cannot detect arrays and instanceof fails across execution contexts (iframes).
